Karya Kami

Project unggulan yang dikembangkan tim Smaridasa Informatics Expo 2026.

🎯
Whack-a-Mole Game
Game interaktif berbasis web — 3 fase dalam 3 menit: Early (mole + Bom A), Mid (tambah Bom B menyamar), Late (Bom A & B random, makin cepat). Global leaderboard via Firebase.
HTML/CSS/JS Firebase Leaderboard
🦾
Excavator Arduino
Miniatur excavator berbasis Arduino yang digerakkan 2 servo motor untuk simulasi gerakan lengan dan bucket. Dikontrol secara manual dengan presisi tinggi.
Arduino C++ 2x Servo
🚪
Pintu Otomatis Arduino
Sistem pembuka pintu otomatis menggunakan sensor ultrasonik HC-SR04. Pintu terbuka secara otomatis saat mendeteksi objek dalam jarak tertentu, lalu menutup kembali.
Arduino Ultrasonic C++
